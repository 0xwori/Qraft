# Tapwise AI Logs

Store Tapwise admin, product, marketing, and planning session notes here.

Use this filename format:

```text
YYYY-MM-DD-HHMM-topic.md
```

Each log should include:
- What was discussed
- What was decided
- What was created or changed
- Any follow-up work

Do not include secrets, tokens, passwords, personal data, customer data, or production data.
