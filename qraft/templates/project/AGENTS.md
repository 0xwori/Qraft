# Project Workspace

This folder is for project-specific context, scripts, assets, tools, outputs, and working notes.

## Rules

- Keep explanations simple and clear.
- Store stable project memory in `Memory.md`.
- Store current context in `context.md`.
- Store helper scripts in `scripts/`.
- Store script metadata in `scripts/manifest.json`.
- Store MCP notes and templates in `mcp/`.
- Store generated files in `outputs/`.
- Store short session notes in `ai-log/`.
- Do not store secrets, API tokens, passwords, personal data, or customer data in markdown or logs.
- Use `.env.example` for variable names only.
