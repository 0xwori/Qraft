# Qraft Core

This folder contains the Qraft product engine.

## Folder Rules

- Keep registries in `registry/`.
- Keep Qraft command scripts in `scripts/`.
- Keep Codex routing skills in `skills/`.
- Keep reusable project starter files in `templates/`.
- Keep shared reusable tools in `tools/`.
- Keep real project workspaces outside this folder in repo-root `projects/`.

## Safety

- Do not store real secrets here.
- Use `.env.example` for variable names only.
- Do not put project deck data inside shared tool source.
